README

Description

The parking barrier gate script draws the gate.

The entry and exit manager scripts show case the user interaction with the system.

The parking ticket script takes care of creating the tickets.

Finally, ticket database script saves the created tickets into the ticket.db to ensure entry and exit validation.

To simulate the parking barrier gate, please run both ParkingEntryManagement.py and ParkingExitManagement.py, and follow the instructions

Enjoy !
